# React + TypeScript + Vite

Ideas:
- Use Relay for implementing Graphql on client-side
- React router to manage routing
- React virtualized for rendering large lists (improvement idea)

Version:
- NodeJS: above 20.x

Run the below command to install all necessary dependencies
```bash
npm install
```
Run the below command to compile the graphql files
```bash
npm run relay
```
Run the below command to start the development server
```bash
npm run dev
```
